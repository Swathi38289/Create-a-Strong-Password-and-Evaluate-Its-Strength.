## ğŸ“ Why Password Length Matters

Password length increases:
- Total possible combinations  
- Time required to crack the password  
- Overall entropy (randomness)  

Long passwords are significantly harder to break than short complex ones.

---

## ğŸ”‘ What is a Passphrase?

A passphrase is a long sentence-like password that is:
- Easier to remember  
- Harder to crack  

Example:
BlueTiger!RunsAcross7Mountains

## ğŸ” What is Multi-Factor Authentication (MFA)?

MFA adds an extra layer of security by requiring:
- Something you know (password)  
- Something you have (OTP / mobile device)  
- Something you are (biometrics)  

Even if a password is compromised, MFA protects the account.
